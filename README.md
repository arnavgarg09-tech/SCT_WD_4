# Smart Task Manager

A modern To-Do Web Application developed using HTML, CSS, and JavaScript.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Complete
- Due Date & Time
- Search Functionality
- Task Filters
- Priority Levels
- Progress Tracking
- Dark Mode
- Local Storage

## Technologies

- HTML5
- CSS3
- JavaScript
